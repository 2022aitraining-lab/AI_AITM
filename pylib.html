Python Library

Comprehensive reference for Python functions and methods

Built-in Functions 69+

print()
Prints the specified message to the screen or other standard output device.
print("Hello, World!") # Output: Hello, World!
len()
Returns the number of items in an object.
len([1, 2, 3]) # Returns 3
type()
Returns the type of an object.
type("hello") # Returns <class 'str'>
range()
Generates a sequence of numbers.
list(range(5)) # Returns [0, 1, 2, 3, 4]
input()
Reads a line from input and returns it as a string.
name = input("Enter name: ") # Prompts user

String Methods 47+

str.upper()
Converts all characters in the string to uppercase.
"hello".upper() # Returns "HELLO"
str.lower()
Converts all characters in the string to lowercase.
"HELLO".lower() # Returns "hello"
str.split()
Splits the string into a list of substrings.
"a,b,c".split(",") # Returns ['a', 'b', 'c']
str.join()
Joins elements of an iterable with the string as separator.
"-".join(["a", "b", "c"]) # Returns "a-b-c"
str.replace()
Replaces occurrences of a substring with another substring.
"hello".replace("l", "L") # Returns "heLLo"

List Methods 11+

list.append()
Adds an element to the end of the list.
lst = [1, 2]
lst.append(3) # lst becomes [1, 2, 3]
list.extend()
Extends the list by appending elements from an iterable.
lst = [1, 2]
lst.extend([3, 4]) # lst becomes [1, 2, 3, 4]
list.insert()
Inserts an element at a specified position.
lst = [1, 3]
lst.insert(1, 2) # lst becomes [1, 2, 3]
list.remove()
Removes the first occurrence of a value.
lst = [1, 2, 3]
lst.remove(2) # lst becomes [1, 3]
list.sort()
Sorts the list in ascending order.
lst = [3, 1, 2]
lst.sort() # lst becomes [1, 2, 3]

Dictionary Methods 11+

dict.get()
Returns the value for a key if it exists, else returns default value.
d = {"a": 1}
d.get("a") # Returns 1
dict.keys()
Returns a view object containing the dictionary's keys.
d = {"a": 1, "b": 2}
d.keys() # Returns dict_keys(['a', 'b'])
dict.values()
Returns a view object containing the dictionary's values.
d = {"a": 1, "b": 2}
d.values() # Returns dict_values([1, 2])
dict.items()
Returns a view object containing the dictionary's key-value pairs.
d = {"a": 1}
d.items() # Returns dict_items([('a', 1)])
dict.update()
Updates the dictionary with key-value pairs from another dictionary.
d = {"a": 1}
d.update({"b": 2}) # d becomes {'a': 1, 'b': 2}

File Operations 8+

open()
Opens a file and returns a file object.
f = open("file.txt", "r") # Opens file for reading
file.read()
Reads the entire content of the file.
content = f.read() # Reads all content
file.write()
Writes a string to the file.
f.write("Hello") # Writes "Hello" to file
file.close()
Closes the file.
f.close() # Closes the file
