HTML Library

Comprehensive reference for HTML elements and attributes

Document Structure 10+

<html>
The root element of an HTML document.
<html lang="en">
<!-- content -->
</html>
<head>
Contains metadata and links to scripts and stylesheets.
<head>
<title>Page Title</title>
</head>
<body>
Contains the visible page content.
<body>
<h1>Hello World</h1>
</body>

Text Elements 15+

<h1> to <h6>
Heading elements from largest (h1) to smallest (h6).
<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>
Defines a paragraph of text.
<p>This is a paragraph.</p>
<span>
Inline container for text.
<span class="highlight">Important</span>
<strong>
Defines important text (bold).
<strong>Bold text</strong>

Links & Media 8+

<a>
Creates a hyperlink.
<a href="https://example.com">Link</a>
<img>
Embeds an image.
<img src="image.jpg" alt="Description">
<video>
Embeds a video.
<video src="video.mp4" controls></video>
